# Contributing to readcraftery-docs

Thank you for your interest in Readcraftery's documentation.

## What this repository accepts

This repository contains public-facing design documents — the GDD, Art Style Guide, Asset Contracts, and Modding Guide. Contributions that improve clarity, fix errors, or improve translations are welcome.

**Accepted contributions:**
- Typo and grammar fixes
- Factual corrections (with justification)
- Translation improvements for the `es/` versions
- Clarifications that don't change design intent

**Not accepted here:**
- New features or design proposals — those are decided by the project author
- Changes to the `en/` documents that contradict decisions already marked ✅ CLOSED in the Architecture Contract
- Content unrelated to the documents in this repo

## How to contribute

Open an issue describing what you found or want to improve. For small fixes (typos, broken links), a pull request is fine directly. For anything that touches design intent, open an issue first so we can discuss before you invest time writing.

This repo covers documentation only. The Readcraftery game engine, official content, and assets are in a separate private repository and are not open for external contributions at this time.
